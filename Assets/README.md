# Aequis Legal — Digital Brand Asset Kit

## Colors
- Navy: `#10233D`
- Gold: `#B98D3E`
- Cream: `#F4EFE4`
- White: `#FFFFFF`

## Typeface
- Wordmark & seal ring text: Liberation Serif (metric match for Times New Roman / a licensed equivalent like Georgia or Freight Text can be substituted for final print production)

## Files

### Logos (transparent background — use on any surface)
- `wordmark-light` — navy "AEQUIS · LEGAL" wordmark, for light backgrounds
- `wordmark-dark` — white "AEQUIS · LEGAL" wordmark, for dark backgrounds
- `primary-logo-light` — full seal/monogram lockup, navy + gold, for light backgrounds
- `primary-logo-dark` — full seal/monogram lockup, white + gold, for dark backgrounds

### Favicons / App Icons (flat background, ready to upload as-is)
- `favicon-light.png/svg` — rounded square, cream background
- `favicon-dark.png/svg` — rounded square, navy background
- `favicon-circle.png/svg` — circular badge, white background, gold ring
- `favicon-circle-dark.png/svg` — circular badge, navy background, gold ring
- `favicon.ico` — multi-resolution (16/32/48/64/128/256px) for website `<link rel="icon">` use

### Practice Area Icons (gold line icons, transparent background)
- `icon-dispute-resolution`
- `icon-corporate-advisory`
- `icon-commercial-litigation`
- `icon-regulatory`
- `icon-investigations`
- `icon-public-law`

## Formats
- `/svg` — vector, infinitely scalable, ideal for web (crisp on all screens), editable in Illustrator/Figma/Inkscape
- `/png` — high-resolution raster exports (3x base size) with transparent backgrounds (except flat favicons), ready for direct use in Word, PowerPoint, websites, email signatures, letterhead, or social profiles

## Usage notes
- Use `-light` variants on cream/white backgrounds and `-dark` variants on navy/dark backgrounds for correct contrast.
- The seal (primary-logo) works well standalone as a letterhead crest, notary-style stamp motif, or social profile image — for the latter, use `favicon-circle` variants instead since they include a filled background.
- For print (letterhead, certificates, signage), hand the `/svg` files to your printer/designer directly rather than the PNGs — vectors will not pixelate at any size.
